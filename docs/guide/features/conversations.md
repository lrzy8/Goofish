# 会话管理

路由：`/conversations`

![会话管理](/screenshots/conversations.png)

实时消息和对话历史管理。

## 功能

### 对话列表

左侧显示所有对话：
- 对方头像和昵称
- 最后一条消息预览
- 最后消息时间
- 未读消息数量

支持加载更多历史对话。

### 消息详情

点击对话后显示完整聊天记录：
- 消息内容
- 发送时间
- 区分自己/对方消息
- 支持加载更多历史消息

### 消息类型

| 类型 | 说明 |
|------|------|
| 文本消息 | 普通文字消息 |
| 图片消息 | 显示图片缩略图 |
| 商品卡片 | 商品分享消息 |
| 订单消息 | 订单相关通知 |

## 实时更新

通过 WebSocket 订阅消息推送：
- 新消息实时显示
- 对话列表自动排序
- 未读数自动更新

点击对话自动标记为已读。

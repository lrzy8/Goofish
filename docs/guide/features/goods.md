# 商品管理

路由：`/goods`

![商品管理](/screenshots/goods.png)

查看账号下的商品列表。

## 功能

### 商品列表

显示商品信息：
- 商品图片
- 商品标题
- 价格
- 状态（在售/已下架）
- 所属账号

### 筛选功能

| 筛选项 | 说明 |
|--------|------|
| 账号筛选 | 选择特定账号查看其商品 |
| 状态筛选 | 按在售/已下架状态筛选 |

筛选条件会保存到本地存储，下次访问自动恢复。

## 数据来源

商品数据从闲鱼 API 实时获取，仅显示在线账号的商品。

::: info 注意
离线账号的商品不会显示，需要先启动账号。
:::

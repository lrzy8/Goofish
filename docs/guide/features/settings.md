# 系统设置

路由：`/setting`

![系统设置](/screenshots/settings.png)

系统配置页面。

## 主题设置

支持三种主题模式：

| 模式 | 说明 |
|------|------|
| 浅色模式 | 白色背景主题 |
| 深色模式 | 深色背景主题 |
| 跟随系统 | 自动跟随操作系统设置 |

主题设置保存到本地存储，下次访问自动应用。

## AI 设置

配置 OpenAI 兼容的 AI 服务，用于自动回复功能。

### 配置项

| 字段 | 说明 | 示例 |
|------|------|------|
| API 地址 | AI 服务的 API 端点 | `https://api.openai.com/v1` |
| API 密钥 | 认证密钥 | `sk-xxx...` |
| 模型名称 | 使用的模型 | `gpt-4o-mini` |
| 系统提示词 | 全局默认提示词 | 见下方 |

### 兼容服务

支持所有 OpenAI 兼容的 API 服务：

- OpenAI 官方
- Azure OpenAI
- 国内代理服务
- 本地部署的 LLM（如 Ollama）

### 系统提示词

全局默认提示词，用于 AI 回复规则：

```
你是一个闲鱼卖家的智能客服助手。
请用简洁友好的语气回复买家的消息。
...
```

### 操作

| 操作 | 说明 |
|------|------|
| 保存设置 | 保存当前配置 |
| 测试连接 | 测试 AI 服务是否可用 |
| 重置提示词 | 恢复为默认提示词 |

::: tip 提示
修改 AI 设置后需要点击「保存设置」才会生效。
:::

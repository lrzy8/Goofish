# 自动回复

路由：`/autoreply`

![自动回复](/screenshots/autoreply.png)

配置自动回复规则。

## 功能

### 规则列表

显示所有回复规则：
- 规则名称
- 匹配类型
- 启用状态
- 优先级

### 匹配类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 精确匹配 | 消息完全等于匹配内容 | 消息 = "你好" |
| 包含关键词 | 消息包含指定关键词 | 消息包含 "价格" |
| 正则表达式 | 使用正则匹配消息 | `/多少钱|价格/` |
| AI 回复 | 调用 AI 生成回复 | 智能理解并回复 |

### 规则配置

| 字段 | 说明 |
|------|------|
| 规则名称 | 规则的标识名称 |
| 启用状态 | 是否启用该规则 |
| 优先级 | 数字越大优先级越高 |
| 匹配模式 | 选择匹配类型 |
| 匹配内容 | 要匹配的文本或正则 |
| 回复内容 | 匹配成功后的回复 |
| 排除匹配 | 勾选后匹配时不回复 |
| 指定账号 | 仅对特定账号生效 |

### AI 回复

选择 AI 回复类型时：
- 可自定义该规则的提示词
- 留空则使用全局提示词
- AI 会根据上下文智能生成回复

## 规则优先级

1. 按优先级数字从大到小排序
2. 优先级相同时按创建时间排序
3. 匹配到第一个规则后停止
4. 排除匹配规则优先处理

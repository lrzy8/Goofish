# 账号管理

路由：`/accounts`

![账号管理](/screenshots/accounts.png)

管理闲鱼账号的核心页面。

## 功能

### 添加账号

1. 点击「添加账号」按钮
2. 粘贴闲鱼 Cookie
3. 填写备注（可选）
4. 提交后自动获取用户信息

::: tip Cookie 获取方法
1. 登录闲鱼网页版
2. 打开浏览器开发者工具（F12）
3. 切换到 Network 标签
4. 刷新页面，找到任意请求
5. 复制 Request Headers 中的 Cookie 值

Cookie 必须包含 `unb` 字段。
:::

### 账号操作

| 操作 | 说明 |
|------|------|
| 启动 | 建立 WebSocket 连接，开始接收消息 |
| 停止 | 断开连接，停止接收消息 |
| 刷新信息 | 重新获取用户昵称和头像 |
| 编辑 | 修改备注信息 |
| 删除 | 断开连接并删除账号数据 |

### 状态显示

- 🟢 在线：WebSocket 已连接
- 🔴 离线：未连接或连接断开
- 显示最后心跳时间

## 实时更新

通过 WebSocket 订阅，账号状态变化时自动刷新列表。

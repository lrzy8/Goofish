# 订单管理

路由：`/orders`

![订单管理](/screenshots/orders.png)

订单查询和发货操作页面。

## 功能

### 订单列表

显示订单信息：
- 订单号
- 商品标题和图片
- 买家信息
- 订单金额
- 订单状态
- 下单时间

### 订单状态

| 状态 | 说明 |
|------|------|
| 获取中 | 正在从闲鱼获取订单详情 |
| 待付款 | 买家未付款 |
| 待发货 | 已付款，等待发货 |
| 待收货 | 已发货，等待确认收货 |
| 交易成功 | 交易完成 |
| 已关闭 | 交易关闭/取消 |

### 筛选功能

- 按账号筛选
- 按状态筛选
- 分页浏览

### 订单操作

| 操作 | 说明 |
|------|------|
| 刷新 | 从闲鱼获取最新订单信息 |
| 发货 | 确认发货（待发货状态可用） |
| 免拼发货 | 虚拟商品免拼发货 |
| 删除 | 删除本地订单记录 |

### 手动获取订单

如果订单未自动同步，可以手动输入：
1. 选择账号
2. 输入订单号
3. 点击获取

## 实时更新

通过 WebSocket 订阅订单变化：
- 新订单自动添加到列表
- 订单状态变化自动更新

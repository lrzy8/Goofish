# 系统日志

路由：`/logs`

![系统日志](/screenshots/logs.png)

查看系统运行日志。

## 功能

### 模式切换

| 模式 | 说明 |
|------|------|
| 实时日志 | 查看当前运行的最新日志 |
| 历史日志 | 查看历史日期的日志文件 |

## 实时日志

显示当前运行的最新日志：

- 自动刷新（3 秒间隔）
- 可手动开关自动刷新
- 显示最新 200 条
- 按级别筛选

## 历史日志

查看历史日志文件：

1. 选择日期
2. 选择日志文件
3. 查看日志内容

显示最多 1000 条记录。

## 日志级别

| 级别 | 颜色 | 说明 |
|------|------|------|
| DEBUG | 灰色 | 调试信息 |
| INFO | 蓝色 | 一般信息 |
| WARN | 黄色 | 警告信息 |
| ERROR | 红色 | 错误信息 |

## 日志格式

```
2025-12-25 10:30:45 | INFO  | Main         | 启动闲鱼卡密机器人...
2025-12-25 10:30:46 | INFO  | WebSocket    | 账号 123456 连接成功
2025-12-25 10:30:47 | WARN  | AutoReply    | 未找到匹配的回复规则
2025-12-25 10:30:48 | ERROR | API          | 请求失败: 网络超时
```

格式：`时间 | 级别 | 模块 | 消息`

## 日志存储

日志文件按日期分目录存储：

```
logs/
├── 2025-12-25/
│   ├── 20251225_103045.log
│   └── 20251225_143022.log
├── 2025-12-24/
│   └── ...
```

每次启动服务会创建新的日志文件。
